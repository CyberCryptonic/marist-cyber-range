# Range Setup Instructions

This section provides step-by-step instructions to build the Marist Cyber Range.

## Build Order (Follow This)

1. Target Environment Setup
2. Red Team Environment Setup
3. SOC Environment Setup

## Structure

- `target-environment/` → enterprise network (AD, file server, users)
- `red-team-environment/` → attacker machines (Kali, jump box)
- `soc-environment/` → monitoring and detection (Security Onion)

## Goal

Follow these instructions in order to fully recreate the cyber range environment using primarily open-source tools.
